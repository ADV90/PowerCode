1) git status - проверка состояния файлов репозиторрия

2) git init - первоначальная инициализация локального репозитория

3) git add "filename" or "." - добавление файла.ов для коммита

4) git commit -m "[task=0001] добавляем описание изменений"- добавляем описание изменений"

5) git remote add origin https://github.com/ADV90/PowerCode.git - подвязываем репозиторий

6) git push - отправка изменений

7) git pull - получение внесенных изменений
